# 安装说明

这份文档是给最终用户的。

适用于：

- 你通过 GitHub Release、网盘或压缩包分发这个扩展
- 用户不通过 Edge / Chrome 扩展商店安装

## 你会拿到什么

通常你会收到一个压缩包，例如：

- `gpt-dialogue-extractor-edge.zip`

先把它解压到本地一个固定目录。

不要直接在压缩包里使用。

## Edge 安装方法

1. 打开 `edge://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择刚刚解压后的扩展目录
5. 安装完成后，打开 ChatGPT 会话页即可使用

## Chrome 安装方法

1. 打开 `chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择刚刚解压后的扩展目录
5. 安装完成后，打开 ChatGPT 会话页即可使用

## 使用方法

### 导出对话

1. 打开一个 ChatGPT 会话
2. 点击页面顶部的 `导出`
3. 选择需要导出的消息
4. 选择导出格式：
   - `导出 JSON`
   - `导出 Markdown`
   - `导出 PDF`
5. 在浏览器保存窗口中选择保存位置

### 使用时间轴

页面右侧会显示用户消息时间轴。

- 悬停刻度可预览消息
- 点击刻度可跳转到对应用户消息
- 当前阅读位置会自动高亮

## 更新方法

如果作者发了新版本：

1. 下载新的压缩包
2. 解压到本地
3. 打开扩展管理页
4. 点击当前扩展的“刷新”

如果目录结构发生变化，最稳妥的方法是：

1. 先删除旧扩展
2. 再重新加载新的解压目录

## 常见问题

### 1. 点击导出没有反应

先尝试：

1. 刷新扩展
2. 刷新 ChatGPT 页面

### 2. 导出 PDF 失败

当前版本的 PDF 是扩展内部生成后下载，不依赖浏览器打印窗口。

如果仍然失败，优先检查：

- 浏览器是否拦截下载
- 是否有安全插件干扰保存流程

### 3. 升级后看起来还是旧版本

这通常是因为：

- 扩展没刷新
- ChatGPT 标签页没刷新

两边都要刷新。
