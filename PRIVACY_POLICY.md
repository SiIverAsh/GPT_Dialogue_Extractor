# Privacy Policy

This extension processes conversation content locally inside the user's browser.

## What the extension accesses

The extension reads the currently opened ChatGPT conversation page in order to:

- collect the visible conversation content
- load older conversation content when the user starts an export
- generate local export files
- support the right-side user-message timeline

## What the extension does with data

The extension uses the conversation data only for local processing in the browser.

It may generate the following local files when the user explicitly starts an export:

- JSON
- Markdown
- PDF

## What the extension does not do

The extension does not:

- send conversation content to a third-party server
- upload conversation content to a remote database
- collect analytics
- sell user data

## Permissions

The extension currently requests:

- `downloads`: used only to let the browser save exported files chosen by the user

## User control

The extension only exports data when the user manually triggers export from the page UI.
